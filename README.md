## This

This is slow library for limited bit integer and float.

## Features

This package have Integer and Float types with bits.

I8, I16, I32, I64

U8, U16, U32, U64

F32, F64

## Getting started

```
dependencies:
  primitive: ^1.0.0
```

## Usage

import dart file waht you want type's name.

```dart
var i8 = I8(0);

var f32 = F32(3.14);

var U64 = U64(BigInt.from(99));
```

U64 is using BigInt class.
